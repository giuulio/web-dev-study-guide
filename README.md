# web-dev-study-guide

In this repo I will build an interactive study guide for web development using React and other skills taught in Codecademy's Full Stack Engineer Career Path. Content is largely based on Codecademy's web dev [cheatsheets](https://www.codecademy.com/resources/cheatsheets/subject/web-development), with clear nuance on outdated content and comments on the field today.

I aim to write all code by hand as an exercise. The notes taken in /content did rely on Claude's help, however.
